# Sağlık Ocağı Takip Sistemi

Bu proje, sağlık ocağı bünyesinde hasta ve randevu takibini sağlamak amacıyla geliştirilmiştir.

## Proje Kapsamı
- Hasta kayıt işlemleri
- Randevu takibi
- Temel veri tabanı işlemleri (ekleme, güncelleme, silme)

## Kullanılan Teknolojiler
- C#
- Oracle Database
- SQL / PL-SQL

## Açıklama
Proje kapsamında veritabanı tasarımı yapılmış, tablolar arası ilişkiler kurulmuş ve
CRUD işlemleri uygulanmıştır. PL/SQL kullanılarak temel trigger ve procedure yapıları
oluşturulmuştur.
